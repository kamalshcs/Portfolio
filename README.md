# Portfolio
A web project focused on building modern, structural components using HTML and CSS.

## Project Components

### 1. Navigation Bar
* Created a responsive menu using Flexbox.
* Includes a logo and four links that stack on smaller screens.
* Added :hover states to make the links feel interactive.

### 2. Project Card
* Built feature cards using the semantic tags.
* Used CSS Grid for the layout of the image, badge, and description.
* Implemented a smooth transition and box-shadow effect when hovering.

### 3. Contact Form
* Designed an accessible form with fields for Name, Email, Subject, and Message.
* Used matching <label> and id attributes for screen readers.
* Added custom :focus styles to highlight active fields for keyboard users.

## Tech Used
* Semantic HTML5 for structure and accessibility.
* CSS3 for layout (Flexbox and Grid) and responsiveness.
